ART IDEA GENERATOR by Phoenix Kane aka occultnix


SYSTEMS USED ------------------------------------------------------------------
Created with SQLiteStudio (3.2.1), Python3, and TKinter library


HOW TO EDIT -------------------------------------------------------------------
To change, add, or remove any colors, adjectives, or emotions, open artfuel.db in an appropriate program (SQLiteStudio recommended) and edit the respective table.

To change the structure of the prompts or the size of the window, open artgenerator.py in any text-editing software. !SPACES ARE IMPORTANT!
